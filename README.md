# 小码书 📕

像刷小红书一样逛 GitHub——开源项目发现产品。

**在线体验**：https://tszmzxxx111xzzzww.github.io/xiaomashu/

## 为什么做这个

GitHub 是"仓库"不是"内容流"：没有推荐信息流、项目预览弱、全英文，对新手门槛高。小码书把热门开源项目变成瀑布流卡片，让"发现好项目"像刷社交软件一样自然。

## 功能

- 发现 / 新秀 / 经典三个频道，语言 + 主题筛选，关键词搜索
- 项目卡片：社交预览图 + 简介 + 标签，点开看 README 速览
- 点赞、收藏、评论（v1 本地存储，先验证需求再上云端）
- 分享到微信/小红书，访问统计，反馈问卷入口

## 技术

纯前端单文件（HTML/CSS/JS），数据来自 GitHub 公开 API，托管于 GitHub Pages，零成本运行。使用 AI 编程工具辅助开发，产品设计与迭代决策由我完成。
